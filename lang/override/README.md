# Custom Language Overrides

This directory (`lang/override/`) allows you to provide custom translations that override or extend the default language files (`lang/*.po`).

## How to use:
1. Create or copy a `.po` file into this directory and give it the same name as the target language you want to override (e.g., `de.po` for German or `en.po` for English).
2. Add the specific translation strings (`msgid` and `msgstr`) you wish to change.
3. You do **not** need to include the entire original file, only the parts you are overriding. The server will automatically load the defaults first and then apply your custom overrides over them.
4. Restart the SPCast Server for the changes to take effect.

## What happens to the defaults?
The original `.po` files provided directly in the `lang/` directory are generated by the server. Even if you accidentally delete them, the server will restore the factory defaults upon startup. Your custom `.po` files inside this `override/` directory remain untouched.
